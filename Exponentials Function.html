Properties of Exponential Functions

-They always pass through the point (0, 1).
-The base determines the rate of growth or decay. 
-They have no upper or lower bound (they grow or decay infinitely for positive or negative x-values, depending on the base).

Understanding Exponential Function

Base greater than 1 (a > 1): The function exhibits exponential growth. As x increases, the function's output increases rapidly. A common example is f(x) = 2^x, where the base is 2.Base between 0 and 1 (0 < a < 1): The function exhibits exponential decay. As x increases, the function's output approaches zero but never reaches it. An example is f(x) = (1/2)^x, where the base is 1/2.Base equals 1 (a = 1): The function becomes a horizontal line at f(x) = 1, regardless of the exponent's value. For instance, f(x) = 1^x.
